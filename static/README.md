# Static UI Preview

Mock-data prototype for the improved dashboard design.

## Open in browser

With server running:

- **Home:** http://127.0.0.1:3002/preview
- **Detail:** http://127.0.0.1:3002/preview/machine?id=lam-yong-shun

Or open `index.html` directly from this folder.

## What's included

- KPI strip (total / running / idle)
- Search + group filter chips
- Improved machine cards with status bar
- Detail page with running hero + date filter on completed jobs table
- Light blue factory theme, DM Sans font
